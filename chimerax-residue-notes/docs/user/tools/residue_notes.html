Residue Notes

The Residue Notes tool is a first-pass ChimeraX interface for reading, browsing, editing, and exporting residue annotations embedded in mmCIF files using the custom _cootnote_residue_note.* loop written by coot_trimmings.

Notes are grouped by residue and can be navigated with the thread list or the previous/next buttons. Export writes an annotated mmCIF by saving the current model as mmCIF and then injecting the custom annotation loop.
